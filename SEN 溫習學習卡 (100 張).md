### 第一部分：融合教育政策與理念
#card 

1. 融合教育的「五大支柱」是什麼？  :: 及早識別、及早支援、全校參與、家校合作、跨界別協作。
    
2. 什麼是「全校參與模式」(Whole School Approach)？ :: 透過全體教職員、學生及家長的配合，建立共融校園文化，滿足所有學生的學習需要。
    
3. 香港保障殘疾學生受教育權利的法例是？ :: 《殘疾歧視條例》(DDO)，要求教育機構作出「合理遷就」。
    
4. 1994 年發表、強調以學生為中心及受教育權利的國際宣言是？ :: 《沙拉曼加宣言》(Salamanca Statement)。
    
5. 融合教育的主要目的之一是什麼？ :: 幫助師生家長認識、接受和尊重個別差異，建構和諧社會。
    
6. 教育局如何向學校提供資源支援 SEN 學生？ :: 按照 SEN 學生人數和問題嚴重程度提供津貼（如學習支援津貼）。
    
7. 為什麼「及早識別」很重要？ :: 透過如小一入學普查等機制，防止學生的問題在求學初期惡化。
    
8. 融合教育的獲益者是誰？ :: 不單是 SEN 學生，還包括普通學生、教師、家長及整個社會。
    
9. 香港目前有針對融合教育的特定立法嗎？ :: 雖然有《殘疾歧視條例》，但目前無獨立的「融合教育法」。
    
10. 學校如何獲得專業意見支援精神問題學生？ :: 教育局與醫管局 7 個分區中心有共識，學校可直接致電尋求諮詢。
    
11. 教育局對手語納入主流課程的立場？ :: 目前無計劃納入課程，但會配合相關小組推廣及提供支援。
    
12. 「合理遷就」在教育情境下的一個例子？ :: 為讀寫障礙學生提供考試延長時間或電腦輔助作答。
    
13. 融合教育與人權的關係？ :: 珍惜及尊重校群中每位成員，保障每位兒童的受教權。
    
14. 融合教育是否會降低學術水平？ :: 它是透過多元化教學策略提升參與度，而非單純降低標準。
    
15. 融合教育中的「跨界別協作」指什麼？ :: 結合教師、教育心理學家、言語治療師及社工等專業力量。
    

## 第二部分：注意力不足/過度活躍症 (ADHD)

16. ADHD 的三大核心特徵是什麼？ :: 分心（難注意細節）、過動（常離座）、衝動（難以等待/插嘴）。
    
17. ADHD 的生理成因通常與什麼有關？ :: 大腦神經傳導物質「多巴胺」失調。
    
18. 針對 ADHD 的教學策略「分拆法」(Chunking) 指？ :: 將長指令或複雜任務拆解成簡短的小步驟。
    
19. 什麼是 ADHD 學生的「合法動體機會」？ :: 在課室中安排如收發作業的小任務，讓其有正當理由走動。
    
20. 管理 ADHD 挑戰行為的常用工具？ :: 代幣制度 (Token System) 和行為契約。
<!--SR:!2026-03-18,1,208-->
    
21. 針對 ADHD 的藥物治療（如 Ritalin）主要作用是？ :: 改善注意力、減低活動量及增強自我控制力。
<!--SR:!2026-03-18,1,208-->
    
22. 服用 ADHD 藥物的常見副作用？ :: 腸胃不適、頭痛、失眠等，通常屬輕微。
    
23. 處理 ADHD 行為問題的優先順序？ :: 先處理情緒，後處理行為。
    
24. 什麼是「有計劃的忽視」(Planned Ignoring)？ :: 只要學生行為不干擾他人且無危險，教師暫時不給予關注以消除誘因。
    
25. 針對 ADHD 的社交訓練重點？ :: 教導替代技巧（如用飲水調整情緒）及耐性訓練。
    
26. 教師給予 ADHD 學生指令的原則？ :: 指令需具體、簡短，最好一次只給一個。
    
27. ADHD 學生在考試中的常見調適？ :: 延長考試時間（通常 25%）、安排獨立考室以減少干擾。
    
28. ADHD 學生表現出的「執行功能」問題？ :: 難以規劃長遠目標、工作記憶弱、自我調節差。
<!--SR:!2026-03-18,1,228-->
    
29. 如何改善 ADHD 學生的工作整理能力？ :: 使用清單、顏色分類文件夾或特定的收拾程序。
    
30. 學校與家長在 ADHD 支援中的角色？ :: 藥物監督與一致的行為獎勵機制之協作。
    

## 第三部分：自閉症類群障礙 (ASD)

31. ASD 的兩大診斷核心特徵？ :: 社交溝通障礙、固定且重複的行為模式/興趣。
    
32. 什麼是「心智解讀能力」(ToM) 弱？ :: 難以理解他人的想法、意圖、信念或情緒。
<!--SR:!2026-03-18,1,208-->
    
33. 什麼是「弱的中央統合能力」(Weak Central Coherence)？ :: 過於注重細節，難以整合訊息以理解整體概念。
    
34. 什麼是 ASD 學生的「執行困難」(Executive Dysfunction)？ :: 難以計劃、靈活切換任務，且常重複錯誤。
    
35. 針對 ASD 的「結構化教學法」(TEACCH) 重點？ :: 提供清晰的視覺提示和規律的課室程序。
    
36. 什麼是「社交故事」(Social Stories)？ :: 透過具體圖文解釋特定社交情境、他人感受及適當反應。
    
37. ASD 學生的感官異常表現？ :: 對聲音、光線或觸覺可能過度敏感或反應遲鈍。
<!--SR:!2026-03-18,1,208-->
    
38. 為何 ASD 學生需要「預告轉變」？ :: 他們依賴常規與預測，突如其來的改變會引發極大焦慮。
    
39. 針對 ASD 學生的視覺提示工具有哪些？ :: 流程圖、視覺時間表、工作清單。
    
40. ASD 學生在語用上的困難？ :: 說話生硬如機器人、難以維持雙向對話、不懂解讀潛台詞。
    
41. 「亞氏保加症」目前在 DSM-5 中如何分類？ :: 已統一納入「自閉症譜系障礙」(ASD)。
    
42. ASD 個案中的「重複性行為」有何功能？ :: 通常是為了自我安撫、減壓或獲取感官回饋。
    
43. 為什麼 ASD 學生適合「圖像思維」？ :: 大腦處理靜態視覺訊息通常比瞬時的口頭訊息更有效。
    
44. 言語治療師如何支援 ASD 學生？ :: 重點在於改善語用能力、社交溝通及非語言表達。
    
45. 在課室如何減少 ASD 學生的焦慮？ :: 建立高度規律性、提供冷靜區、減少環境雜音。
    

## 第四部分：特殊學習困難 (SpLD/讀寫障礙)

46. SpLD 的核心缺陷通常在哪方面？ :: 語音處理缺損 (Phonological processing deficit)。
    
47. 讀寫障礙學生的典型表現？ :: 認讀速度慢、字體左右倒轉、默書極差、朗讀時漏字。
<!--SR:!2026-03-18,1,208-->
    
48. 針對中文字書寫的調適策略「寫大字」作用是？ :: 幫助學生看清筆畫細部結構和部首的空間分佈。
    
49. 多感官學習法 (Multisensory) 如何應用？ :: 同時運用看、聽、讀、寫（如用手在不同質感的表面畫字）。
    
50. 什麼是「容許以錄音代替手寫」？ :: 讓口頭表達較好的學生能繞過書寫障礙展現其學科知識。
    
51. 為讀寫障礙學生設計考卷的原則？ :: 字體較大、行距較寬、少用長題目，提供關鍵詞庫。
    
52. 中文字形分析的教學重點？ :: 部首概念、左支右絀的偏旁組合、筆順規律。
<!--SR:!2026-03-18,1,190-->
    
53. 如何提升讀寫障礙學生的閱讀理解？ :: 使用視覺組織圖、心智圖、將長文章分段。
    
54. 讀寫障礙學生一定智力較低嗎？ :: 不。SpLD 學生通常智力正常，甚至有部分非常優秀。
    
55. 「左右倒轉」寫字在什麼年紀後仍頻繁出現才需警惕？ :: 小三 (Primary 3) 之後若持續頻繁出現則需留意。
    
56. 讀寫障礙學生默書的調適方法？ :: 減少默書範圍、改默核心詞彙、提供填充式默書。
    
57. 辨別相似字（如：未/末）的方法？ :: 用顏色標示不同之處或使用口訣/聯想法。
    
58. 考試調適中的「讀題服務」適用於？ :: 嚴重讀寫障礙學生，幫助其理解題目內容而非測試其認字。
    
59. 為什麼讀寫障礙學生常被誤解為「懶散」？ :: 因為他們口才可能很好，但遇到文字作業時卻顯得極度困難。
    
60. 常見的書寫筆順規律？ :: 先撇後捺、由上至下、由左至右、先中間後兩邊。
    

## 第五部分：智力障礙 (ID)

61. 智力障礙的診斷準則？ :: 智商 (IQ) 顯著低於常人（通常 70 以下）且適應行為（如自理）有困難。
    
62. 什麼是「工作分析法」(Task Analysis)？ :: 將複雜任務（如洗手）拆成極微小、可執行的單一小步驟。
    
63. ID 學生適合哪種課程類型？ :: 功能性課程 (Functional Curriculum)，強調生活實用技能。
<!--SR:!2026-03-18,1,208-->
    
64. 教導 ID 學生抽象概念時應注意？ :: 具體化原則，使用實物、模型或直接生活體驗。
    
65. 什麼是「社區實務教學」？ :: 帶領學生到真實環境（如超市、銀行）練習社交與生活技能。
    
66. ID 學生的「類化能力」(Generalization) 弱是指？ :: 在一個情境學會的技能，難以自動轉移到另一類似情境。
    
67. 針對 ID 的行為策略「積極強化」？ :: 完成一小步即給予立即、具體的讚賞或獎勵。
    
68. ID 學生的 IEP 指什麼？ :: 個別化教育計劃 (Individualized Education Program)。
<!--SR:!2026-03-18,1,208-->
    
69. 為何要讓 ID 學生學習「自我決策」？ :: 幫助他們在未來能對自己的生活有一定的選擇權與責任感。
    
70. ID 學生在記憶方面的特徵？ :: 工作記憶容量較小，需高頻率的複習與過度練習。
    
71. 教學內容的調適三層次？ :: 內容 (Content)、過程 (Process)、成果 (Product)。
    
72. 「分層教學」的一個例子？ :: 同一課題，A組計算應用題，B組計算純數字題，C組進行數數。
    
73. ID 學生的教學目標應如何設定？ :: 小步子、循序漸進，並與日常生活緊密結合。
<!--SR:!2026-03-18,1,208-->
    
74. 職能治療師 (OT) 在 ID 支援中的作用？ :: 訓練精細動作、手眼協調及日常自理能力。
    
75. 智力障礙學生能讀主流學校嗎？ :: 輕度 ID 學生在獲得適切支援的情況下可於主流學校就讀。
    

## 第六部分：資優與雙重特殊 (Gifted/2e)

76. 什麼是「雙重特殊」(Twice-Exceptional / 2e)？ :: 同時擁有資優特徵與一或多種 SEN 障礙（如 ADHD 或讀寫障礙）。
    
77. 2e 學生的教學調適重點？ :: 採納「雙軌並行」，既要發展優勢潛能，也要針對障礙提供遷就。
    
78. 導致資優學生「潛能未展」的原因？ :: 缺乏挑戰性、自我要求過高、缺乏成就動機或社交孤立。
    
79. 校本資優培育的三層架構（第一層）？ :: 在全班式教學中加入高層次思維、創造力及情意教育。
    
80. 資優生的典型行為特質？ :: 學習速度極快、好奇心強、具領導潛能、有獨創性。
    
81. 支持資優生的多元化評估？ :: 非語言智力測驗、行為量表、家長提名及作品評鑑。
    
82. 如何激發資優生的學習動機？ :: 提供開放式問題、研究計畫，而非重複性的練習。
    
83. 資優生的社交挑戰？ :: 可能感到與同輩「不同」，導致社交焦慮或顯得孤傲。
    
84. 什麼是差異化教學的「等化器維度」？ :: 課程調整的九個刻度，如：具體到抽象、簡單到複雜。
    
85. 「等化器」中「基礎 vs 轉換」的分別？ :: 基礎指核心概念；轉換指將概念應用到全新的、不熟悉的情境。
    

## 第七部分：語言、感官與精神健康

86. 語言障礙 (SLI) 學生的表達特徵？ :: 發音不清、句子結構破碎、詞彙量匱乏、難以敘述事件。
    
87. 什麼是「空泛詞」問題？ :: 在溝通中頻繁使用「那個、這件」，顯示語義檢索有困難。
    
88. 與言語障礙學生溝通的原則？ :: 保持眼神接觸、給予充足回應時間、說話速度要適中。
    
89. SLI 學生的課業調適？ :: 容許運用較短句子或圖示回應，提供句式支架 (Sentence frames)。
    
90. 精神病 (MI) 在學生中的常見類型？ :: 焦慮症、抑鬱症、對立性反抗症。
<!--SR:!2026-03-18,1,208-->
    
91. 香港青少年最盛行的精神障礙（研究數據）？ :: 焦慮症及抑鬱症，對立性反抗症亦值得關注。
<!--SR:!2026-03-18,1,208-->
    
92. 抑鬱症學生的外顯表現？ :: 長期情緒低落、對興趣失去熱情、睡眠障礙、體重劇變。
    
93. 視覺障礙 (VI) 學生的環境調適？ :: 提供充足光線、避免障礙物、使用放大教材或盲文。
    
94. 聽覺障礙 (HI) 學生主要依賴哪種訊息？ :: 視覺訊息（如字幕、手語、讀唇及視覺提示）。
    
95. 學校如何應對學生的「對立性反抗」(ODD)？ :: 建立正面的師生關係，避免硬碰硬，設定清晰一致的獎懲。
<!--SR:!2026-03-18,1,208-->
    
96. 精神健康問題學生的轉介機制？ :: 透過學校小組，在徵得家長同意後與醫生或專業人士協作。
    
97. 教學等化器中「獨立程度」的維度？ :: 從「多教師引導」逐步過渡到「低教師引導」。
<!--SR:!2026-03-18,1,208-->
    
98. 什麼是「思考等化器」中的「小跳躍 vs 大跳躍」？ :: 指從已知訊息推論到未知結論的跨度與難度。
    
99. SEN 學生的「代幣制度」成功關鍵？ :: 獎勵必須是學生渴望的，且需立即與特定行為掛鉤。
    
100. 融合教育的核心價值觀？ :: 尊重多樣性，確保每個孩子都能在最少限制的環境下學習。
