## 第 1 課：電腦系統基礎

### 1. 桌上電腦的基本部件
- 傳統桌上電腦由「系統組」和「周邊設備」組成。
- 系統組 (System Unit)
  - 是一個機箱，容納電腦的主要部件。
  - 內部包含：主機板、中央處理器 (CPU)、主記憶體 (RAM)、輔助儲存設備 (SSD/硬碟)、電源供應器。
- 周邊設備 (Peripherals)
  - 連接到系統組的硬件設備。
  - 輸入設備：鍵盤、滑鼠、麥克風、網絡攝錄機。
  - 輸出設備：顯示器、打印機、揚聲器。
  - 儲存設備：便攜式硬碟、USB 手指。

### 2. 系統組內部結構
- 主機板 (Motherboard)
  - 一塊電路板，擁有多個插座或插槽。
  - 用於安裝 CPU、RAM、SSD 和硬碟。
  - 設有擴展槽 (Expansion Slots) 用於裝配額外的電路板 (如顯示卡)。

- 系統匯流排 (System Bus)
  - 讓數據可以在 CPU 和其他部件之間傳輸。
  - 分為三大類型：數據匯流排、位址匯流排、控制匯流排。

## 第 2 課：中央處理器 (CPU) 和圖形處理器 (GPU)

### 1. 中央處理器 (CPU)
- 電腦的「腦袋」，負責運行程式並處理數據。
  - 透過系統匯流排與其他設備通訊。
  - CPU 的運算能力決定電腦整體性能。
- 電腦工作的三個階段：輸入 → 處理 → 輸出。
- 機器週期 (Machine Cycle)：提取 (Fetch) → 譯碼 (Decode) → 執行 (Execute)。

### 2. CPU 的重要組件
- 算術及邏輯部件 (ALU)
  - 執行算術運算 (加減乘除)。
  - 執行邏輯運算 (AND, OR, NOT, 數據比較)。
- 控制部件 (CU)
  - 從記憶體獲取指令並解譯 (Decode)。
  - 指示 ALU 進行運算。
  - 控制輸入、輸出和儲存設備。
- 快取記憶體 (Cache)
  - 位於 CPU 內的小型高速記憶體。
  - 儲存經常使用的數據，減少從主記憶體 (RAM) 讀取的時間。
  - 分為 L1, L2, L3 多層，可提升 CPU 性能約 20%。

---
### 3. 性能指標
- 時鐘頻率 (Clock Speed)：以 MHz 或 GHz 為單位。1 Hz = 每秒 1 個週期。
- 核心數量 (Cores)：多核心可同時處理不同任務或並行處理，提升效能。

### 4. 圖形處理器 (GPU) 與 NPU
- GPU (Graphics Processing Unit)
  - 專門設計用於高速處理圖像和並行運算。
  - 擁有數千個核心 (CPU 通常只有幾個)，適合渲染 3D 圖像、視像編輯。
  - 處理高密度計算任務時比 CPU 快很多。
- NPU (Neural Processing Unit)
  - 神經網絡處理器，專用於人工智能 (AI) 和機器學習。
  - 例子：Apple Neural Engine, Google TPU。
  - 速度單位：TOPS (Tera Operations Per Second)。

## 第 3 課：主記憶體 (RAM 和 ROM)

### 1. 主記憶體概覽
- 安裝在主機板上的記憶體晶片，儲存 CPU 執行的數據和指令。
- 主要分為 RAM (隨機存取記憶體) 和 ROM (唯讀記憶體)。

### 2. 隨機存取記憶體 (RAM)
- 易失性 (Volatile)：關機後數據會消失。
  - 暫時儲存使用中的軟件數據。
- 可讀寫：數據隨電腦運行不斷改變。
- 容量單位：GB (如 8GB, 16GB, 64GB)。
- 字長 (Word Size)
  - 現今多為 64 位元 (64-bit)。
  - 字長越大，CPU 搜尋和存取數據越快。
- 速度計算公式：數據存取速度 = 字長 × RAM 速度 (MHz)。

### 3. 唯讀記憶體 (ROM)
- 非易失性 (Non-volatile)：關機後數據不會消失。
  - 永久儲存製造商提供的資料。
- 用途：儲存啟動程式 (Boot Loader) 和 BIOS (基本輸入輸出系統)。
- 功能：電腦啟動時，從 ROM 載入啟動程式，再載入硬碟中的操作系統 (如 Windows)。

---

### 4. 快閃記憶體 (Flash Memory)
- 由 ROM 轉化而成，非易失性，但可讀寫。
  - 例子：記憶卡 (SD Card)、USB 手指、固態硬碟 (SSD)。
- SSD 已逐漸取代傳統機械硬碟，成為主要儲存設備。
