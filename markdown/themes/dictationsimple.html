📋 Quick Reference:

  • H1 (#): Main exam title - centered with borders
  • H2 (##): Section headings - left aligned with bottom border
  • H3 (###): Subsection headings - for question categories
  • .mc-question: Multiple choice container
  • .mc-options: Auto-numbered A, B, C, D options
  • .correct: Add to correct option (makes it bold)
  • .answer-line: Shows answer with "答案：" prefix
  • .long-question: Long answer question container
  • .answer-space: Creates answer lines (165mm width)
  • .instruction: Italic instructions/notes
  • ---: Page break in Marp
