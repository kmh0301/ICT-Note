# 第六課 網絡安全 溫習筆記

### I. 網絡欺凌 (Cyberbullying) 及 起底 (Doxxing)

#### 網絡欺凌
網絡欺凌會使使用者即時受到恐嚇、羞辱、虐待、或感到難堪、被針對。

| 行為 | 影響 | 預防/處理 |
| :--- | :--- | :--- |
| 透過電郵或社交網站**散播謠言**。 | 出現健康問題（如失眠）。 | **發言前細思：** 體諒別人想法，避免侮辱或專罵別人。 |
| 發布他人**難堪的影片和照片**。 | 出現心理問題（如抑鬱）。 | **尊重他人的私隱：** 發布照片/訊息須先得到他人允許。 |
| 在未經對方同意下，於網上公開對方的**真實資料**（如姓名、地址、電話等）。 | 容易受毒品和酒精的誘惑。 | 處理方式：向社交平台或機構管理員舉報；尋求教師、家長或社工等協助。 |

#### 起底 (Doxxing)
起底指從不同地方（如論壇或社交媒體帳戶）收集目標人物或其相連人士的資訊，並在互聯網上公開。
*   **影響：** 為受害人帶來巨大的壓力，使受害人持續處於恐懼之中。
*   **後果：** 根據2021年條例，起底會被視為**刑事罪行**。刑罰可能包括罰款十萬港元及監禁兩年；嚴重情況下，可能被罰款一百萬港元並監禁五年。

### II. 惡意軟件 (Malware) 及種類

惡意軟件指能夠**盜取資料**、**損壞電腦裝置或網絡**的程式。

| 種類 | 定義/特性 | 自我複製 | 依附其他程式 | 自動在網絡傳播 |
| :--- | :--- | :--- | :--- | :--- |
| **電腦病毒** (Virus) | 依附於可執行檔和文件與試算表巨集，用戶開啟後感染。 | **✘** | **✔** | **✘** |
| **木馬程式** (Trojan Horse) | 假裝成一些**實用的應用程式**。一旦開啟，黑客即可盜取用戶資料。 | **✘** | **✔** | **✘** |
| **蠕蟲** (Worm) | 能夠**自我複製**，並在電腦網絡中**自動傳播**，能在短時間內令大量裝置受感染。 | **✔** | **✘** | **✔** |
| **勒索軟件** (Ransomware) | 黑客用作**加密**受害人系統數據的惡意軟件，要求支付贖金以解密。 | N/A | N/A | N/A |
---
### III. 預防惡意程式的方法

為防止黑客入侵及惡意軟件感染，用戶應採取以下措施：

1.  **安裝防毒軟件：** 應定期更新防毒軟件，以確保持能對抗最新的惡意程式。防毒軟件可掃描感染檔案和檔案。
2.  **安裝防火牆：** 用戶應為裝置安裝防火牆。
    *   **功能：** 能夠**監控電腦網絡**，過濾外來的**惡意網絡數據包**。
    *   **傳播阻止：** 透過防止對外傳輸，阻止蠕蟲在電腦網絡傳播。
    *   **形式：** 防火牆可以是**軟件或硬件裝置**。
    *   **注意：** 防火牆**不可**掃描電腦上的檔案以查找病毒。
3.  **定期更新電腦系統和程式：** 可預防黑客利用未更新的漏洞入侵操作系統。

### IV. 社交工程 (Social Engineering)

社交工程指透過**誘惑或恐嚇**，從而取得個人的資料。

| 技巧名稱 | 定義 | 例子 |
| :--- | :--- | :--- |
| **餌誘** (Baiting) | 誘餌方式主要利用受害者的好奇心。故意**遺下帶有惡意程式的儲存裝置**（如USB記憶棒/光碟），以感染拾獲該裝置的受害人電腦。 | 故意遺下裝有惡意程式的USB記憶棒 [8, 49c]。 |
| **假託** (Pretexting) | 透過**虛構故事情節**，誘騙受害人提供個人資料或從中獲利。 | 發送電郵給受害者，並聲稱自己已綁架其兒子以騙取贖金。 |
| **仿冒詐騙** (Phishing) | **冒充網上銀行網站**，以騙取受害人的私人資料（如銀行和信用卡資料）。 | 建立一個與一般網上銀行相似的網站，從而騙取受害人的戶口資料。 |
| **非社交工程技巧** | **入侵**不是社交工程的技巧。 | N/A |

### V. 勒索 (Extortion)

勒索罪犯法造成或威脅受害者，從受害者身上取得利益。

| 勒索方式 | 定義 |
| :--- | :--- |
| **網絡擄掠勒索** (Cyber-blackmail) | 散播多個互聯網帳號使用者的資訊；威脅將受害者倒楣的照片或片段拍下來，用以勒索。 |
| **勒索軟件** (Ransomware) | 惡意軟件加密系統數據，要求受害者支付贖金解密。 |
---
### VI. 網上不實資訊 (Untrue Information Online)

網上不實資訊是指由任何人在網上發布的資訊，但我們卻錯誤地相信網上資訊是準確且真實的。

*   **預防方法：**
    *   在發布或傳播資訊前，應**檢查資訊是否來自可靠的來源**。
    *   檢視資訊的可靠性，例如進行**事實查核 (fact-check)**。



#### 核心重點概念對比 (是非題總結)

1.  **社交工程 vs 惡意軟件：** 利用電腦病毒入侵他人的電腦**並非**社交工程。勒索軟件**並非**社交工程。
2.  **蠕蟲 vs 木馬程式：** 蠕蟲能在電腦網絡**自動傳播**，但木馬程式**不能**。
3.  **防火牆功能：** 防火牆**不可**掃描電腦上的檔案以查找病毒，但它可以是軟件或硬件裝置。
4.  **網上欺凌：** 在互聯網上公開他人的個人資料是網上欺凌的一種。
