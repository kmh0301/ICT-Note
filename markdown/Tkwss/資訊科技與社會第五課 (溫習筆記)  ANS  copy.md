# 第五課 知識產權 溫習筆記

## I. 知識產權 (Intellectual Property, IP) 概述

知識產權旨在賦予創作者（個人、群體或公司）的權利。尊重知識產權能為藝術家和發明者提供**安全公平的環境**，讓他們能夠安心創作。

### 知識產權的種類與保障

| 權利名稱 | 定義與保障範圍 |
| :--- | :--- |
| **版權 (Copyright)** | 自動授予作品**原創者**的權利。版權保障原創作品的作者。 |
| **專利權 (Patent)** | 保障**發明品**，授予擁有者防止他人未經授權複製的權利。如果你發明了技術想禁止他人抄襲，應申請專利。 |
| **外觀設計權** | 對產品**外觀**的保障。 |
| **商標權 (Trademark)** | 對分辨不同商戶的**貨品標誌**的保障。**請注意：** 商標權**無法**防止任何未經授權使用原創作品的行為。 |

## II. 版權 (Copyright) 與受保護作品

### 1. 版權保障的範圍
*   版權是**自動授予**作品原創者的權利，**不須註冊**。
*   香港的**《版權條例》**包含保護版權持有人的條例，以防止侵權行為。
*   受版權保護的作品包括：
    *   **文學作品：** 例子有小說、電腦程式、詩歌、文章。
    *   **戲劇、音樂或藝術作品。**
    *   **錄音、廣播、電影或電視節目。**
    *   **出版刊物的排印編排。**
    *   軟件的**源碼**為版權所保障。
---
### 2. 版權擁有人的權利
版權擁有人在香港享有專屬權利，包括：
*   製作、發布、散播、改編及翻譯作品。
*   授權他人使用受版權保護的作品。
*   出售你的版權。
*   透過遺囑中將版權繼承給他人。


## 共享創意 (Creative Commons) 的四個授權條件

共享創意授權條款允許創作者在保留部分權益的情況下，讓他人可以分發和使用他們的創作。

| 條件名稱 | 縮寫 | 定義與要求 |
| :--- | :--- | :--- |
| **署名** | BY | 授權他人可以複製、分發、展示和演出作品，以及創作衍生改作的作品。但使用者必須依照**創作者指定的署名方式**進行署名。 |
| **相同方式共享** | SA | 授權他人發放衍生改作的創作。但規定衍生作品的版權條款必須與**原作的授權條款相同**。 |
| **非商業性** | NC | 授權他人可以複製、分發、展示和演出作品，以及創作衍生改作的作品。但所有這些使用都**只限於非商業用途**。 |
| **禁止衍生** | ND | 授權他人可以複製、分發、展示和演出**原作**。但**不得改動或衍生改作**該作品。 |

## III. 軟件特許使用權 (Software Licensing)

軟件授權決定了用戶如何使用、限制或重新發放軟件。

#### 1. 四種主要的軟件授權

| 種類 | 定義/特性 | 價格/費用 | 源碼存取 | 技術支援 | 例子 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **商業軟件** | 具有廣泛支援的商業軟件。通常需支付特許使用權費用。 | 須付費用。 | 不可以。 | 最多支援。 | Microsoft Word |
| **共享軟件** | 提供免費試用版，但有時間或功能限制，可能需付註冊費。**可能**有試用期。 | 免費試用，但可能須付錢購買註冊金鑰。 | 不可以。 | 可以。 | WinZip。 |
| **免費軟件** | 可供用戶**免費下載及使用**。**沒有**試用期。 | 免費。 | 不可以。 | 有限或無支援。 | WhatsApp、瀏覽器 (如 Microsoft Edge)。 |
| **開放源碼軟件** | 可供用戶**免費下載及使用**，用戶可對軟件作出**修改**，亦可**自由發放**軟件。 | 免費。 | **可以**，可自由更改、複製、發放。 | Linux、Open Office。 |

### 2. 關於共享軟件和免費軟件的不同之處
*   **共享軟件**或有試用期，而**免費軟件**則沒有。
*   **共享軟件**或允許用戶發布，而**免費軟件**則可能有限制。

## IV. 侵權行為與預防

### 1. 網上盜版 (Online Piracy)
*   **定義：** 指在互聯網上**未經授權地取得或分發**受版權保護的作品。
*   **例子：** 在網上下載未經授權的電影是網上盜版的一種。
*   **後果：** 侵犯版權的行為會被視為刑事罪行，可能被判罰款或監禁。

### 2. 引用與抄襲
*   抄襲指盜用他人的作品或想法。
*   在引用資料時，標明來源是**非常重要**的。
*   **請注意：** 即使是為了研究目的，我們也**不可以**隨意使用互聯網上的資料。

### 3. 侵權帶來的影響
*   **對創作者：** 如果沒有得到適當的酬勞，創作者可能會失去動力創造新穎和高質素的設計。
*   **對國家：** 盜版活動對全球經濟造成影響，可能使國家蒙受損失。
