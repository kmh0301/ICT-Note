# 內容
二維碼是一種二維條碼，經常用於各種海報、書籍、網站、應用程式中。

二維碼由排列在白色方形背景中的無數黑點組成。它表示一組字串，通常是一個劃一資源定位（Uniform Resource Locator，簡稱 URL）、一個電話號碼或一個電子郵件地址。
