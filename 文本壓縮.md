# 內容
**文本**包括各種類型的字符,如數字、英文字母、中文字符、符號等。  
**文本**可分為兩種類型:  
* **純文字**：只能**儲存字符**。
* **格式化文本**：能儲存字及其格式資料,如字型種類、字體大小、  
字體顏色、行距、水平對齊方式等,其大小相對比純文字大。

| ====    | **純文字**                              | **格式化文本**                                                |
| --- | ------------------------------------ | -------------------------------------------------------- |
| 功能  | 只能**儲存字符**                           | 能儲存字符及其格式資料,如字型種類、字體大小、  <br>字體顏色、行距、水平對齊方式等,其大小相對比純文字大。 |
