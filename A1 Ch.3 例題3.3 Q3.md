# ◣ 分析
情況 (1)，如果 Q = 0，則 P + Q = 7，不會超出無符號整數的範圍。

情況 (2)，由於 P、Q、R 均為無符號整數，它們只能是零或正數。將零或正數加至 P 的話， 應會得出比 P 更大的整數。

情況 (3)，如果 P > 3 和 Q > 4，則 P + Q > 7。一個 3 位元的無符號整數不可大於 7。
