# Python 程式設計基礎 (第二章) - 學生教學手冊

這份手冊將進入程式設計中非常強大的 **循環 (Loop)** 概念。學會循環後，你可以讓電腦自動重複執行成千上萬次的動作。


## Lesson 1: 基礎迴圈 (for loop)

### 📚 教學重點 (Summary)
當你需要電腦重複做某件事時，不需要手寫很多次同樣的代碼，可以使用 **for 迴圈 (for loop)**。
*   **for 迴圈 (for loop)**：這是一個「遍歷」的過程。想像你有一個盒子，裡面裝了很多個數字，`for` 指令會依次把裡面的東西拿出來，執行一次動作。
    *   **語法**：
        ```python
        for i in [1, 2, 3, 4]:
            print("hi, Python")
        ```
    *   **i (控制變數)**：這是一個暫時的變數，每次循環它都會代表列表中的一個值。
    *   **工作原理**：在上面的例子中，列表中有 4 個元素，所以 `print("hi, Python")` 會被重複執行 **4 次**。

> [!TIP]
> 就像 `if` 語句一樣，`for` 語句的結尾必須有 **冒號 (:)**，且循環內部的程式碼必須 **縮進 (Indentation)**。

### 🎯 任務描述 (Task)
編寫一個程式，使用 `for` 迴圈遍歷列表 `[1, 2, 3, 4]`，並在每次循環時顯示 "hi, Python"。

### 💻 程式碼模版 (Template)
```python
# 1. 使用 for 迴圈遍歷 [1, 2, 3, 4]
for i in [1, 2, 3, 4]:
    # 2. 顯示輸出內容
    
```

### ✅ 參考答案 (Solution)
```python
for i in [1, 2, 3, 4]:
    print("hi, Python")
```

### ✍️ 額外挑戰 (Exercise)
**題目**: 請修改程式，讓它顯示 5 次 "Hello World"。

**挑戰答案**:
```python
for i in [1, 2, 3, 4, 5]:
    print("Hello World")
```

## Lesson 2: range() 函數的使用

### 📚 教學重點 (Summary)
如果你想重複執行 100 次，手寫一個包含 100 個數字的列表太累了。這時我們需要 **range() 函數**。
*   **range(start, stop)**：這會生成一個數值序列。
    *   **start (起始值)**：序列開始的數字。
    *   **stop (終點值)**：序列在那裡停止（注意：**不包含** 這個數字本身）。
    *   **例子**：`range(1, 101)` 會生成從 `1` 到 `100` 的數字序列。
*   **配合 for 迴圈**：
    ```python
    for i in range(1, 101):
        print(i)
    ```
    這會讓電腦從 1 數到 100，並按順序把每個數字顯示出來。

> [!IMPORTANT]
> 記住 `range(1, 10)` 只會數到 9！如果你想數到 10，必須寫成 `range(1, 11)`。

### 🎯 任務描述 (Task)
編寫一個程式，使用 `for` 迴圈和 `range()` 函數，顯示從 `1` 到 `100` 的所有整數。

### ✅ 參考答案 (Solution)
```python
for i in range(1, 101):
    print(i)
```

### ✍️ 額外挑戰 (Exercise)
**題目**: 使用 `range()` 顯示從 `10` 到 `20` 的數字（包含 20）。

**挑戰答案**:
```python
for i in range(10, 21):
    print(i)
```

## Lesson 3: 累加運算 (Accumulation)

### 📚 教學重點 (Summary)
迴圈不僅能顯示數字，還能幫我們做數學統計。最常見的就是 **累加 (Summation)** 模式。
*   **累加器 (Accumulator)**：我們通常會先定義一個變數（例如 `sum = 0`）來當作「存錢筒」。
*   **更新變數**：在迴圈內部，我們使用 `sum = sum + i`。這表示把當前的數字 `i` 加進存錢筒裡，然後更新存錢筒的總數。
*   **邏輯拆解**：
    1.  設定 `sum = 0`。
    2.  迴圈開始，`i` 分別為 1, 2, 3...
    3.  每次都執行 `sum + i` 並存回 `sum`。
    4.  迴圈結束後，顯示 `sum` 的最終結果。

> [!WARNING]
> 注意 `print()` 的位置！如果你把 `print()` 放在縮進裡面，它會每次加法都顯示一次結果；如果你放在縮進外面（左對齊），它只會在最後顯示一個最終總數據。

### 🎯 任務描述 (Task)
編寫一個程式，計算從 `1` 加到 `100` 的總和，並輸出結果。

### 💻 程式碼模版 (Template)
```python
# 1. 創立一個存放總和的變數
total_sum = 0

# 2. 使用 for 迴圈與 range 遍歷 1 到 100


# 3. 在迴圈後顯示最終結果
```

### ✅ 參考答案 (Solution)
```python
total_sum = 0
for i in range(1, 101):
    total_sum = total_sum + i

print(total_sum)
```

### ✍️ 額外挑戰 (Exercise)
**題目**: 計算從 `1` 直到 `50` 的總和。

**挑戰答案**:
```python
s = 0
for i in range(1, 51):
    s = s + i
print(s)
```
