# 內容
## 定義
**試算表軟體**是方便我們處理數據的應用程式。它可以協助用戶完成不同工作，如進行複雜的計算、以圖表呈現數據和處理表格形式的數據等。使用這套軟體令我們可以更簡單地處理數據，並減少分析數據所需的人力。

試算表軟件具備多項功能以方便數據處理，例如：
• 公式與預設函數；
• 篩選與排序；
• 設定格式化條件；以及
• 圖表。

試算表軟件的例子有 Microsoft Excel、Google 試算表與 LibreOffice Calc。
