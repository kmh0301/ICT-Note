文本包括各種類型的字符，如數字、字母、符號等。儲存在電腦中時，它們由字符編碼系統表示。不同語言的文本可能會使用不同的編碼系統。

| 字符編碼系統 | 語言   | 字符示例 |      |
| ------ | ---- | ---- | ---- |
| ASCII  | 英文   | 1234 | ABCD |
| Big5   | 繁體中文 | 一二三四 | 甲乙丙丁 |
文本可分為兩種類型：
- 純文字只會儲存字符

- 格式化文本會儲存字符及其格式資料，如字型、字體大小、字體顏色、行距、對齊方式等，其檔案大小比純文字大
