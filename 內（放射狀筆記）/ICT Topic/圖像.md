# 內容
可分為兩類：
- [[點陣圖]]
- [[向量圖]]

數碼圖像是由數碼相機捕捉或使用圖像編輯軟件製作的圖像

| 種類  | 優勢                                          |
| --- | ------------------------------------------- |
| 點陣圖 | 寫實數碼照片通常是點陣圖， 可以用常見的圖像編輯軟件（如Microsoft小畫家）編輯 |
| 向量圖 | 檔案大小通常較小，沒有粗糙的邊緣，在放大或縮小時不會變得模糊              |

圖像檔案有四項重要的屬性
1. [[解像度]]
2. [[色深]]
3. [[長寬比]]
4. [[透明度]]

## 檔案大小的計算：
未壓縮的點陣圖檔案大小為:
$$色深\times解像度=色深\times寬度(像素)\times高度(像素)$$
向量圖和經壓縮的點陣圖大小則較難估計。
