# 內容
用以表示數據的方法有很多。 
• 以物理方式用連續值表示的數據（如聲波、文字和圖畫）稱為模擬數據。
• 以 0 和 1 等離散符號表示的數據（如文字檔案、數碼照片和音頻檔案）則稱為數碼數據。
